# Release v0.0.18

## 🐛 Bug Fixes

### Include main worktrees, exclude submodules, preserve glob patterns (#96)

Fixed several issues with global worktree discovery and config path handling:

- **Include main worktrees in global discovery**: `gwq list -g` now detects repos where `.git` is a directory (main worktree), not just linked worktrees where `.git` is a file
- **Exclude submodules**: Submodule `.git` files are now detected and skipped, preventing submodule entries from appearing in `gwq list -g`
- **Preserve glob patterns in config**: `repository_settings` entries containing glob characters (`*?[`) are no longer broken by `filepath.Abs()` prepending the CWD
