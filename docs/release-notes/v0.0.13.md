### `ls` Alias for `list` Command (#74)

Added `ls` as a short alias for the `list` command, providing a familiar UX similar to `tmux ls`.

**Usage:**

```bash
# These commands are equivalent
gwq list
gwq ls

# All flags work with the alias
gwq ls -v
gwq ls --json
gwq ls -g
```
