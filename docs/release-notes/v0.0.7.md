### Glob Pattern Matching for Repository Settings (#50)

- Added `doublestar` library for `**` recursive glob pattern support
- Repository settings paths now expand `~` correctly with `ExpandPath`
- `copy_files` option supports `**` patterns for recursive file matching
- Added comprehensive tests for glob pattern matching

Example:

```toml
[[repository_settings]]
repository = "~/src/myproject"
copy_files = ["templates/**/*.env"]  # Now supports ** patterns
```
