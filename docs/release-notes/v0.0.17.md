# Release v0.0.17

## 🐛 Bug Fixes

### Support git URLs using SSH config aliases (#93)

Fixed `gwq add` failing when the git remote uses an SSH config alias without `git@` prefix (e.g., `workgit:myorg/myrepo.git`).

- Added `isSCPLikeURL()` helper to detect SCP-like syntax and convert `host:path` to `https://host/path` for URL parsing
- Distinguishes SCP-like aliases from `host:port` URLs by checking if the segment after `:` is numeric
